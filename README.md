# vue-i18n-utils

Collection of utility components for vue-i18n-manager

## Installation

```bash
$ npm install vue-i18n-utils
```

## Usage

```js
import Vue from 'vue'
import VueI18nUtils from 'vue-i18n-utils'

Vue.use(VueI18nUtils)
```
